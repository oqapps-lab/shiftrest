## Flow 1: First Launch (Первый запуск)

**Цель:** Новый пользователь проходит от установки до получения первого персонализированного плана сна.

**Решает:** F6 (Onboarding), F1 (Schedule Input), F2 (Sleep Plan)

### Happy Path

```
[Установка из App Store / Google Play]
    ↓
Welcome Screen (ценностное предложение + CTA «Создать мой план»)
    ↓
Шаг 1: Профессия — выбор: медсестра / пожарный-EMT / фабрика / другое
    ↓
Шаг 2: Тип графика — шаблон ротации (3x12, 24/48, continental) или «своё»
    ↓
Шаг 3: Текущая смена — день/ночь, время начала/конца
    ↓
Шаг 4: Главная проблема — «не могу заснуть после ночной» / «переход ночь→день» / «хроническая усталость»
    ↓
[Social Proof Break — «93% сменных работников не высыпаются» + отзыв коллеги]
    ↓
Шаг 5: Хронотип — 3 вопроса из MEQ (сова/жаворонок/голубь)
    ↓
Шаг 6: Кофеин — сколько чашек/день, чувствительность (normal / slow)
    ↓
Шаг 7: Мелатонин — принимаете ли, дозировка
    ↓
Шаг 8: Семья — дети, фиксированные обязательства (напр. «дети из школы в 15:00»)
    ↓
Шаг 9: Имя — «Как вас зовут?» (+13% конверсии на paywall)
    ↓
[Social Proof Break — отзыв от медсестры/пожарного по выбранной профессии]
    ↓
Loading Screen — «Анализируем ваш профиль сна...» (3-5 сек, анимация)
    ↓
★ Aha-момент — «[Имя], вот ваш план сна»
    Timeline: смена (серый) → дорога (жёлтый) → сон (синий) → мелатонин (фиолетовый) → caffeine cutoff (красный)
    Превью transition plan
    ↓
Paywall — персонализированный заголовок, 4 буллета ценности, 2 тарифа, trial timeline, «Может быть позже»
    ↓
  ├── [Начал trial] → Notification Permission Screen → Home (с планом)
  └── [«Может быть позже»] → Notification Permission Screen → Home (ограниченный план)
```

**Метрики:**
- Onboarding completion: целевой 60%+ (median H&F: 26%, [PRACTICES-BRIEF.md](../03-practices/PRACTICES-BRIEF.md))
- Trial start rate: целевой 10%+ (median: 6.5%)
- Время прохождения: ≤2 минуты

### Error States

| Ситуация | Поведение |
|----------|-----------|
| Нет интернета на Welcome | Offline-banner: «Нет подключения. Для создания плана нужен интернет». Retry-кнопка |
| Нет интернета на Loading | Локальный fallback: rule-based план без AI. Баннер: «Offline-план. Подключитесь для полного анализа» |
| Ошибка создания плана (API) | Fallback на rule-based алгоритм. Пользователь не видит ошибку |
| Paywall не загрузился (Adapty) | Пропуск paywall → сразу Home. Paywall покажется позже контекстно |
| Пользователь закрывает приложение на шаге 5 | При следующем открытии — продолжение с шага 5 (прогресс сохранён) |

### Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Профессия «Другое» | Дополнительный вопрос: «Опишите ваш график» → ручной ввод смен |
| Нет готового шаблона ротации | Переход к ручному вводу: календарь + выбор дат |
| Пользователь пропускает шаги (skip) | Заполняем defaults: средний хронотип, 2 чашки кофе, без мелатонина, без семьи. Баннер на Home: «Дополните профиль для точного плана» |
| Повторная установка (reinstall) | Свежий онбординг. Данные не сохранены без аккаунта |

## Flow 2: Core Action — Просмотр и следование плану сна

**Цель:** Пользователь видит сегодняшний план сна и следует рекомендациям в реальном времени.

**Решает:** F2 (Sleep Plan), F4 (Caffeine Cutoff), F5 (Melatonin Timing), F7 (Notifications)

### Happy Path

```
[Открытие приложения / Push-уведомление «Пора готовиться ко сну»]
    ↓
Home — Сегодняшний план:
    • Sleep window: 08:30 - 16:00
    • Caffeine cutoff: 01:00 (через 3 часа — зелёный)
    • Melatonin: 07:45 (через 8 часов)
    • Следующая смена: завтра 19:00
    ↓
Tap на Sleep Plan карточку → Sleep Plan экран
    ↓
Visual Timeline: 24-часовая полоса
    Смена (19:00-07:00) → Дорога (07:00-07:30) → Wind-down (07:30-08:30) → Сон (08:30-16:00) → Свободное время → Смена
    ↓
Карточки рекомендаций:
    • ☕ Caffeine Cutoff: «Последний кофе — до 01:00» [подробнее]
    • 💊 Мелатонин: «Примите 1-3 мг в 07:45» [подробнее]
    • 🕶️ Свет: «Наденьте тёмные очки на выходе» [подробнее]
    • 😴 Nap: «Pre-shift nap 17:30-18:15 (45 мин)» [подробнее]
    ↓
[Push: «Последний кофе — сейчас» в 01:00]
    ↓
[Push: «Наденьте тёмные очки» в 06:45]
    ↓
[Push: «Примите мелатонин» в 07:45]
    ↓
[Push: «Пора готовиться ко сну» в 08:00]
    ↓
[Сон по плану] → Утро: Sleep Streak +1 🔥
```

### Error States

| Ситуация | Поведение |
|----------|-----------|
| Нет расписания на сегодня | Home: «Добавьте смену на сегодня» → переход в Schedule |
| AI-план недоступен (API error) | Fallback: rule-based план. Визуально идентичен AI-плану |
| Notifications отключены в системе | Баннер на Home: «Включите уведомления, чтобы не пропустить мелатонин» → Deep link в системные настройки |
| Нет интернета | Кэшированный план (последний синхронизированный). Баннер: «Offline — показан последний план» |

### Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Пользователь работает переработку (+4ч) | Кнопка «Я задержался» → пересчёт плана с новым временем конца смены |
| Выходной день (нет смены) | Home показывает: «Сегодня выходной 🎉», рекомендация по восстановлению режима, Sleep Streak |
| Между блоками смен (transition) | Автоматическое предложение Transition Plan (Flow 5) |
| Первый день использования | Home: Aha-момент план из онбординга. Подсказка: «Свайпните для деталей» |

## Flow 3: Return Visit (Повторное открытие)

**Цель:** Вернувшийся пользователь за <30 секунд видит, что ему нужно сейчас.

**Решает:** F2 (Sleep Plan), F7 (Notifications), retention

### Happy Path

```
[Push-уведомление ИЛИ привычка открыть приложение]
    ↓
Home — мгновенная загрузка:
    ┌──────────────────────────────┐
    │  Привет, Марина! 🔥 4 дня   │  ← Sleep Streak
    │                              │
    │  ⏰ Сейчас: Caffeine OK ☕🟢  │  ← Контекстная карточка
    │  Cutoff через 2ч (в 01:00)  │
    │                              │
    │  Следующее:                  │
    │  07:45 — Мелатонин 💊       │
    │  08:30 — Время сна 😴       │
    │                              │
    │  [Смотреть полный план →]    │
    └──────────────────────────────┘
    ↓
  ├── Глянул → закрыл (80% сессий — <1 мин)
  ├── Tap «Полный план» → Sleep Plan экран
  └── Tap на Schedule tab → просмотр/редактирование расписания
```

### Error States

| Ситуация | Поведение |
|----------|-----------|
| Расписание закончилось (не продлено) | Home: «Ваше расписание закончилось. Продлить на 4 недели?» → Schedule |
| Долго не открывал (>14 дней) | Welcome-back экран: «С возвращением! Обновите расписание» → Quick schedule update → Home |
| Подписка истекла | Home: базовый план (ограниченный). Баннер: «Разблокируйте полный план» → Paywall |

### Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Открытие в 3 AM на смене | Контекстная карточка: «Вы на смене. Caffeine cutoff — в 01:00 (прошёл ⚠️)» |
| Открытие сразу после ночной смены | Контекстная карточка: «После смены: мелатонин в 07:45, сон с 08:30» |
| Выходной день, утро | Контекстная карточка: «Выходной! Сон прошлой ночью: OK ✅. Transition plan →» |

## Flow 4: Upgrade Flow (Бесплатный → Платный)

**Цель:** Free-пользователь видит ценность premium и подписывается.

**Решает:** Monetization, F8 (AI Coach), F3 (Transition Plan — premium)

### Happy Path

```
[Бесплатный пользователь на Home]
    ↓
Видит базовый план (sleep window только) — без мелатонин timing, caffeine детально, transition
    ↓
Tap на «Transition Plan» карточку (или Melatonin, или AI-рекомендации)
    ↓
🔒 Premium Feature Gate — preview:
    «Ваш план перехода ночь→день: 3 дня, 7 шагов»
    Показано: первые 2 шага (размыто) + CTA «Разблокировать»
    ↓
Contextual Paywall:
    Заголовок: «[Имя], разблокируйте Transition Plan»
    4 буллета про transition planning
    2 тарифа + trial timeline
    «Может быть позже»
    ↓
  ├── [Начал trial] → Unlock → Transition Plan полный → 🎉 Celebration
  ├── [«Может быть позже»] → Возврат к бесплатному Home
  └── [Повторный показ через 3 дня ИЛИ при следующем tap на premium фичу]
```

**Точки показа paywall:**
1. **Онбординг** (основной) — 50% конверсий в Day 0
2. **Tap на premium-фичу** (контекстный) — Transition Plan, AI Coach, Melatonin Timing
3. **Конец первого рабочего блока** (proactive) — «Блок смен завершён! Transition plan поможет восстановиться»
4. **Settings** → «Управление подпиской» (passive)

### Error States

| Ситуация | Поведение |
|----------|-----------|
| Ошибка оплаты (Apple/Google) | «Не удалось обработать оплату. Проверьте данные карты» + кнопка retry |
| Adapty недоступен | Paywall не показывается. Логирование ошибки. Повтор при следующем триггере |
| Пользователь в trial, но отменил | Доступ до конца trial period. Баннер: «Trial заканчивается через 2 дня» |
| Restore Purchase не находит подписку | «Подписка не найдена. Убедитесь, что используете тот же Apple/Google аккаунт» + Support link |

### Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Пользователь видел paywall 3+ раз и не подписался | Снижаем частоту: не чаще 1 раза в 7 дней. Меняем контекст/буллеты |
| Trial закончился, не подписался | «Ваш trial завершён» → soft downgrade. Базовый план остаётся. Premium фичи locked |
| Пользователь подписался через App Store напрямую | Adapty синхронизирует. Premium разблокируется автоматически |

## Flow 5: Transition Planning (Переход между режимами)

**Цель:** Пользователь получает и проходит пошаговый план перехода ночь→день (или день→ночь).

**Решает:** F3 (Transition Plan) — killer feature, главный дифференциатор

### Happy Path

```
[Конец блока из 3 ночных смен — автоматическое определение]
    ↓
Push: «Блок ночных завершён! Ваш план перехода на дневной режим готов 🌅»
    ↓
Tap → Transition Plan экран:
    ┌─────────────────────────────────────┐
    │  Переход: Ночной → Дневной режим    │
    │  3 дня • 7 шагов • Старт: сегодня  │
    │                                     │
    │  День 1 (сегодня):                  │
    │  ✅ 07:00 — Тёмные очки на выходе   │
    │  ⬜ 07:45 — Мелатонин 1 мг          │
    │  ⬜ 08:00 — Короткий сон (4-5ч)     │
    │  ⬜ 13:00 — Подъём, яркий свет      │
    │  ⬜ 15:00 — Дети из школы            │
    │  ⬜ 22:00 — Мелатонин 0.5 мг        │
    │  ⬜ 23:00 — Сон (цель: 7-8ч)        │
    │                                     │
    │  День 2 (завтра): [свёрнуто →]      │
    │  День 3: [свёрнуто →]               │
    │                                     │
    │  [💡 Почему именно так?]            │
    └─────────────────────────────────────┘
    ↓
Пользователь следует шагам, отмечая выполненные (checklist)
    ↓
Push-уведомления на каждый шаг:
    07:00 — «Наденьте тёмные очки 🕶️»
    07:45 — «Примите мелатонин 💊»
    08:00 — «Ложитесь спать. Будильник на 13:00»
    13:00 — «Подъём! Откройте шторы ☀️»
    22:00 — «Мелатонин перед сном 💊»
    ↓
День 3: Полный переход завершён
    ↓
🎉 Celebration: «Вы успешно перешли на дневной режим! Sleep Streak: 4 🔥»
    ↓
Home обновляется: дневной режим, следующий блок смен через X дней
```

### Error States

| Ситуация | Поведение |
|----------|-----------|
| Нет интернета при создании плана | Кэшированный template-based план (без AI). Баннер: «Базовый план. Подключитесь для персонализированного» |
| Пользователь не отмечает шаги | Push-напоминание через 30 мин. Не навязчиво — 1 retry, потом пропуск шага |
| AI-план содержит невалидное время | Validation layer: все времена проверяются rule-based. Невалидные → замена на default |
| Нотификации отключены | Баннер на Transition Plan: «Включите уведомления для напоминаний о каждом шаге» |

### Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Смена сменилась (подмена / overtime) | Кнопка «График изменился» → пересчёт Transition Plan с нового времени |
| Пользователь уже «испортил» первый день (спал не по плану) | «Ничего страшного! Адаптируем план» → пересчёт оставшихся дней |
| Переход день→ночь (обратный) | Отдельный шаблон: advance sleep gradually, тёмные очки вечером, свет утром |
| Семья: дети из школы в 15:00 | План учитывает: сон 08:00-14:30, подъём за 30 мин до 15:00 |
| Быстрый переход (1 день между блоками) | «Экспресс-план»: сокращённая версия на 1 день вместо 3 |
